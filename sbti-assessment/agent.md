
# AI 阅读指引：SBTi 评估网站

## 项目背景

该项目为 `sbti` 下的企业气候目标成熟度评估页面，目标是为企业提供基于 SBTi 框架的在线评估工具。页面采用单文件 HTML 结构，包含样式、交互逻辑和结果展示。

## 关键文件

- `sbti_assessment_website.html`
  - 主页面文件
  - 包含评估题目数据、前端交互逻辑、结果评估和可视化展示
  - 同时包含广告和统计脚本

## 任务优先级

1. 保持评估逻辑完整，避免改变题目评分规则
2. 优化页面可访问性与移动端体验
3. 美化视觉风格，使页面更专业、简洁、现代
4. 优化 SEO 元数据、社交分享标签和结构化数据
5. 提升页面可维护性，必要时适当整理 CSS 与 JavaScript

## 修改范围

- 允许修改：
  - `sbti_assessment_website.html`
- 不建议修改：
  - 其他无关目录中的文件
  - 非业务相关的外部统计脚本（除非有明确优化需求）

## 额外说明

- 如果需要添加组件或新的样式块，请尽量在页面内使用语义化标签（例如 `header`、`main`、`section`、`footer`）
- 如果要修改交互逻辑，确保保留“上一题/下一题”、“进度条”、“结果分析”功能
- 对于 SEO 优化，请优先检查页面标题、描述、OG 标签、Twitter 卡片、结构化数据和可访问性
