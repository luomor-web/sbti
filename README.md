# SBTI 人格测试网站

## 项目介绍

SBTI (Student's Big Five Traits Inventory) 人格测试网站是一个基于大五人格理论的在线测试工具，旨在帮助学生了解自己的人格特质。

## 技术栈

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome

## 项目结构

```
├── index.html          # 首页
├── test.html           # 测试页面
├── results.html        # 结果页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── main.js         # 主脚本
│   ├── test.js         # 测试逻辑
│   └── results.js      # 结果处理
└── README.md           # 项目说明
```

## 功能特点

1. 基于大五人格理论的测试题目
2. 直观的用户界面
3. 即时的测试结果分析
4. 响应式设计，支持多种设备

## 如何使用

1. 打开 `index.html` 进入首页
2. 点击「开始测试」按钮进入测试页面
3. 回答所有测试题目
4. 提交后查看测试结果

## 测试维度

1. 开放性 (Openness)
2. 责任心 (Conscientiousness)
3. 外向性 (Extraversion)
4. 宜人性 (Agreeableness)
5. 神经质 (Neuroticism)
