# SBTI AI 读取说明

## 项目概述

SBTI 人格测试网站是一个基于大五人格理论的在线测试工具，面向学生用户。

该项目通过静态页面提供测试、结果展示和分析功能，适合作为轻量级前端演示或学习项目。

## 技术栈

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome

## 关键文件说明

- `index.html`：项目首页，提供测试入口。
- `test.html`：测试页面，包含题目和提交逻辑。
- `results.html`：结果页面，展示测试分析结果。
- `css/style.css`：样式文件。
- `js/main.js`：主脚本，包含页面交互逻辑。
- `js/test.js`：测试题目处理逻辑。
- `js/results.js`：结果计算与展示逻辑。
- `README.md`：项目说明文档。

## 目标

- 让 AI 快速理解项目结构和功能定位。
- 为后续文档改进、代码维护或功能扩展提供参考。
- 支持 AI 读取并生成更准确的建议、说明或开发方案。
